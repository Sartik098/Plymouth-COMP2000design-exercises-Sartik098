## Overview
This repository contains the Android Studio implementation of the Staff User Interface for the Restaurant Management Mobile Application. This work forms part of COMP2000 Assessment 1 (Design Exercises) and Assessment 2 (Implementation).

The app supports high-fidelity UI layouts for staff features, including:
- Login
- Dashboard
- Manage Menu
- Add/Edit Menu Items
- View Reservations
- Reservation Details
- Notifications
- Settings

No Java/Kotlin backend logic is included, as per Assessment 1 requirements. The layouts are prepared to support future integration with the COMP2000 REST API.
